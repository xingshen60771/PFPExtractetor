### 基本介绍
本工具用于演示PFP格式的游戏资源文件的解压缩功能，理论上讲可以解压缩游戏厂商PlayFirst开发或发行所有格式为PFP的游戏资源包文件，同时基于该厂商的PlaygroundSDK开发的其他游戏也可解压。本软件演示了PFP文件算法的文件偏移、大小、文件目录信息的存储方式和数据区的内容存储规则，对研究资源逆向基础提供了重要参考资料。
